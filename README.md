# AIMHOUSE

Repositório de estratégia, análises e relatórios da operação Tamara Damares.

## Estrutura

- `estrategia/` — relatórios de estratégia e análise de performance
  - `relatorio-criativos-imagem.html` — performance histórica dos 24 criativos de imagem (formato AD) das campanhas de Captação Paga

## Como visualizar

Abra os arquivos `.html` diretamente no navegador, ou ative GitHub Pages no repositório para acesso via URL pública.
